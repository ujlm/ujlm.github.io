A simple solution to convert currencies directly in your HTML code.
The source code can be found on Github.

Basic example

                    
                        <span
                        class="xratejs"
                        from='[{"USD": 1}]'
                        to="EUR"
                        loading...</span>
Result
$1 equals €loading....

Total portfolio example

                    
                        <span
                        class="xratejs"
                        from='[{"BTC": 0.5}, {"GBP": 20}, {"EUR": 10}]'
                        to="JPY"
                        </span>
                        
                        <span
                        class="xratejs"
                        from='[{"BTC": 0.5}, {"GBP": 2}, {"EUR": 10}]'
                        to="USD"
                        </span>
Result
Bro, my super balanced investment portfolio is worth ¥ !
That's € !
I'm so cool bro!
